앱스스크립트 제작 실습

🎙️ AI 상담 자동기록 앱 핵심 실습 아래 단계에 따라 API 키를 발급받고 앱을 배포해보세요.
시트 복사하기 API Studio 바로가기
Step 1. API 키 발급

1. Get API key 클릭 → 2. API 키 만들기 클릭 → 3. 키 만들기 클릭

Step 2. API 키 입력 (Apps Script)
  • 상단 탭 '확장 프로그램 → Apps Script'로 이동합니다.
  • index.html 파일에서 Ctrl+F를 눌러 api를 찾아 " " 사이에 본인의 API 키를 입력 후 저장합니다.
const apiKey = "🟩🟩🟩이곳에 복사한 API 키를 복붙해주세요.(초록 네모도 삭제)🟩🟩🟩";
Step 3. 웹 앱 배포
  • 우측 상단 배포 → 새 배포를 클릭합니다.
  • 유형 선택: 웹 앱
  • 액세스 권한: 모든 사용자

배포 완료 후 생성된 웹 앱 URL을 복사하여 접속하세요!
